## 示例<a name="zh-cn_topic_0104202532_zh-cn_topic_0102902530_section17446171164041"></a>

-   请求样例：

    ```
    {
        "new_owner": "scuser1",
        "group_name": "groupName"
    }
    ```

-   成功响应样例：

    ```
    {
        "is_success": "true",
        "message": ""
    }
    ```

    >![](public_sys-resources/icon-note.gif) **说明：**   
    >调用接口出错后，将不会返回上述结果，而是返回错误码和错误信息，详细介绍请参见[错误码](错误码.md)。
