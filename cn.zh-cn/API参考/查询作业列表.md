# 查询作业列表<a name="dli_02_0234"></a>

## 功能介绍<a name="section123211931103215"></a>

查询当前用户的作业列表，可以根据作业ID作为ID，查询大于ID或小于ID的限定条数的作业，默认查询全部状态的作业，也可以设定运行中或其他状态条件。

## URI<a name="section18631191318335"></a>

-   URI格式
