## 示例<a name="section17446171164041"></a>

-   请求样例：
    -   **描述:**

        批量删除项目ID为48cc2c48765f481480c7db940d6409d1队列名称为queue1，扩缩容计划ID为3和4的定时扩缩容计划。

    -   **示例URL:**

        POST https://\{endpoint\}/v1/48cc2c48765f481480c7db940d6409d1/queues/queue1/plans/batch-delete

    -   **值:**

        ```
        {
           "plan_ids": [3,4]
        }
        ```


-   成功响应样例：

    ```
    {
        "is_success": true,
        "message": ""
    }
    ```
