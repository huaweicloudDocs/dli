    ```
    None
    ```

    ```
    {
        "create_time": 1572867705362,
        "update_time": 1572867705362,
        "resource_type": "jar",
        "resource_name": "luxor-router-1.1.1.jar",
        "status": "READY",

        "owner": "user1"
    }
    ```
