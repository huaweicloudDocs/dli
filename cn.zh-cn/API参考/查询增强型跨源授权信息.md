-   请求样例：

    ```
    None
    ```


-   成功响应样例:

    ```
    {
        "is_success": true,
        "message": "",
        "privileges": [
            {
                "object": "edsconnections.503fc86a-5e60-4349-92c2-7e399404fa8a",
                "applicant_project_id": "330e068af1334c9782f4226acc00a2e2",
                "privileges": [
                    "BIND_QUEUE"
                ]
            }
        ],
        "connection_id": "503fc86a-5e60-4349-92c2-7e399404fa8a"
    }
    ```
