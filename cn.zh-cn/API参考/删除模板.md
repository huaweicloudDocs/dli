# 删除模板<a name="dli_02_0247"></a>

## 功能介绍<a name="s89ff8bc59cba4c3b94dc17e85c8fa1ea"></a>

-   URI格式

    DELETE /v1.0/\{project\_id\}/streaming/job-templates/\{template\_id\}
