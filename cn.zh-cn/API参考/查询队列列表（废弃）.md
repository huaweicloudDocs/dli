-   请求样例：

    ```
    None
    ```

-   成功响应样例：

    ```
    {
      "is_success": true,
      "message": "",
      "queues": [
        {
         "charging_mode":1,
         "create_time": 1508143955000,
         "cu_count": 4,
         "description": "test",
         "owner": "tenant1",
         "queue_name": "queue1"
        }
      ]
    }
    ```

    >调用API出错后，将不会返回上述结果，而是返回错误码和错误信息，详细介绍请参见[错误码](错误码.md)。
